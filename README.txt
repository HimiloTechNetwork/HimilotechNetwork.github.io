# ZeroFour Test Template

Tani waa tijaabo template ah oo laga keenay [HTML5 UP](https://html5up.net). Waxaa loogu talagalay in lagu barto qaab-dhismeedka responsive web design.

## Features
- Responsive layout
- Multilevel dropdown menus
- Font Awesome icons
- jQuery + Dropotron integration
